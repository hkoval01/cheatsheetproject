HTML: Lists and Unordered Lists

Unordered List Tags

Tag Name Description
<ul> Unordered List A list of things, normally in bullet points, but attributes can change the shape; e.g. <ul type='square'>
<li> List Item The individual points in an unordered/ordered list

<ul> Example:

<ul type ='square'>

<li>List item</li>

</ul> =

  • List item

Ordered List Tags

Tag Name Description
<ol> Ordered List A list of things by number, but the type attribute can change the numbers to letters or roman numerals; e.g. <ol type='a'>
<li> List Item The individual points in an unordered/ordered list

<ol> Example:

<ol type ='a'>

<li>List item</li>

</ol> =

  1. List item
